> 一个为 LLM API 调用设计的轻量级网关，通过智能路由、语义缓存和多上游故障转移，**降低 Token 成本**，同时保证模型透明不降级。

## ✨ 产品特色

- **🔥 显著降本**：内置语义缓存、缓存池化、输出长度限制，实测减少大量 Token 消耗。
- **🤖 多模型统一接入**：支持 OpenAI、Claude、Gemini、通义千问、豆包等主流模型，完全兼容 OpenAI API 格式。
- **🛡️ 模型透明**：不降级、不偷换模型，调用日志可追溯。
- **⚡ 高可用**：多上游故障自动切换，可用性 99.9% 以上。
- **🧩 轻量易用**：一行 Base URL 即可接入，无需修改业务代码。
